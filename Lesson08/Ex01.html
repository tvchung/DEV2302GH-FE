Cú pháp tương tác của Js với HTML

            - inline (thuộc tính sự kiện của phần tử)
            - internal (viết lệnh xử lý trong cặp thẻ script)
            - external (viết lệnh js trong file .js)

inline (thuộc tính sự kiện của phần tử)

internal (viết lệnh xử lý trong cặp thẻ script)

- Nên đặt trong phần head - Tuy nhiên có thể viết cuối trang, trước thẻ đóng body

external (viết lệnh js trong file .js)

- Viết mã lệnh javacript trong file .js - Nhúng file js vào html thông qua thẻ script - Nên đặt cuối trang, trước thẻ đóng body
