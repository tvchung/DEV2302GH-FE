Cú pháp tương tác của DOM với các phần tử HTML

            1. Tương tác thông qua thuộc tính id
                document.getElementById 
            2. Tương tác thông qua tên thẻ HTML 
                document.getElementsByTagName 
            3. Tương tác thông qua thuộc tính class của phẩn tử HTML 
                document.getElementsByClassName 
            4. Tương tác thông qua css selector 
                document.querySelectorAll
